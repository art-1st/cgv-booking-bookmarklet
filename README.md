# CGV 좌석 스나이퍼 Bookmarklet

CGV 영화 예매 시 이미 매진된 인기 좌석의 **취소표/환불표를 자동으로 감지하여 점유**해주는 JavaScript Bookmarklet입니다.

## 동작 원리

1. **좌석 후보 선택** — 전체화면 좌석맵에서 잡고 싶은 좌석을 우선순위 순으로 선택 (매진 좌석 포함)
2. **자동 감시** — 페이지 내 새로고침 버튼을 주기적(2초)으로 클릭하며 좌석 가용 여부 체크
3. **자동 점유** — 후보 좌석 중 빈 자리가 발견되면 즉시 좌석 클릭 → 선택완료 → 서버 점유
4. **알림** — 비프음 + 진동 + 화면 알림으로 사용자에게 통지

## 사용 방법

### 1. Bookmarklet 등록

`bookmarklet.min.js` 내용 앞에 `javascript:void(` 를 붙여 북마크 URL에 등록합니다.

> **모바일**: 임의 페이지를 북마크 → 편집에서 URL을 교체

### 2. 실행

1. CGV에서 영화, 극장, 일시, 회차를 선택하여 **좌석 선택 페이지** (`selectVisitorCnt`)에 진입
2. Bookmarklet 실행
3. 자동으로 일반 1명 선택 → 전체화면 좌석맵 열림 → **좌석 후보 선택 UI** 표시
4. 잡고 싶은 좌석을 **우선순위 순서대로** 터치/클릭
   - 빨간색 = 매진 좌석, 파란색 = 선택 가능 좌석
   - 선택된 좌석에 순번 표시, 다시 터치 시 해제
5. **[확인 - 감시 시작]** 클릭
6. 하단 패널에 감시 상태 표시 (새로고침 횟수)
7. 빈 자리 발견 시 자동 점유 → 소리 + 진동으로 알림
8. 결제를 진행하세요!

### 감시 중지

하단 패널의 **[중지]** 버튼을 누르면 감시가 종료됩니다.

## 파일 구조

```
bookmarklet.js      # 원본 소스 (주석 포함)
bookmarklet.min.js  # minified 버전 (bookmarklet 등록용)
README.md           # 사용 설명서
```

## 설정

`bookmarklet.js` 상단의 상수를 수정하여 동작을 조절할 수 있습니다:

| 상수 | 기본값 | 설명 |
|------|--------|------|
| `POLL_INTERVAL` | `5000` (ms) | 새로고침 기본 간격 |
| `POLL_JITTER` | `1000` (ms) | ±랜덤 범위 (실제 간격: 4~6초) |

## 주의 사항

- **1인 일반(성인) 예매 전용**입니다.
- CGV 웹사이트 구조 변경 시 동작하지 않을 수 있습니다.
- 극장/상영관별 좌석 배치가 달라도 DOM 클래스 기반 동적 수집으로 대응합니다.
- 여러 좌석을 후보로 등록하면 우선순위가 높은 좌석부터 점유를 시도합니다.
- 동일 좌석을 다른 사용자가 먼저 점유할 수 있으므로, 점유 실패 시 자동으로 다음 후보를 시도합니다.
